# Unemployment Analysis with Python

## Description
This project analyzes unemployment rate data in India using Python. The analysis focuses on identifying trends, seasonal patterns, and the impact of COVID-19 on unemployment rates through data visualization.

## Dataset
- Unemployment in India.csv
- Contains unemployment rate percentages by region and date

## Technologies Used
- Python
- Pandas
- Matplotlib

## How It Works
1. Load and clean the dataset
2. Analyze unemployment trends
3. Visualize results using graphs
4. Interpret insights

## Goal
To understand unemployment patterns and gain insights useful for economic and social policy decisions.
